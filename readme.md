# Codewars Solutions

Dépôt regroupant mes solutions aux katas [Codewars](https://www.codewars.com), réalisés dans le cadre de ma progression en algorithmie et en JavaScript.

## Objectifs

- Consolider ma maîtrise des méthodes JavaScript (`map`, `filter`, `reduce`, `Set`, etc.)

- Documenter ma progression de manière transparente

## Organisation

Les solutions sont classées par thème. Chaque fichier contient la solution accompagnée d'un commentaire expliquant l'approche adoptée.

```
codewars-solutions/
├── arrays/
├── strings/
├── objects/
├── math/
└── mixed/
```
